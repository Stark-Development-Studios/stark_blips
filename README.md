# Stark Blips

## Description
Stark Blips Is A Standalone Blip Creator Script For FiveM.

## Installation
1. Add stark_blips to your resources folder for your server
2. Be sure to remove the '-main' from the end of the folder name
3. Customize the script to your liking in the Config!
4. Add ```ensure stark_blips``` to your server.cfg
5. Restart your server and enjoy!

## Features
1. Blip Creation To Mark Useful Points Of Interest On The Native GTA V Map.

## Supported Frameworks
1. Qbox
2. QBCore
3. ESX
4. Standalone

## Dependency
1. [ox_lib](https://github.com/overextended/ox_lib/releases)
